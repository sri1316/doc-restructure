# Enterprise AI Documentation (Rspress)

This repository contains the documentation site for the Enterprise AI platform, built with Rspress.

Getting started

1. Install dependencies:

   npm install

2. Start dev server:

   npm run dev

3. Build production bundle:

   npm run build

4. Preview production build locally:

   npm run preview

Notes:

- Requires Node.js and npm/yarn/pnpm installed. If you plan to edit TypeScript files, keep `typescript` installed as devDependency.
- The site root is `docs/` and the Rspress config is `rspress.config.ts`.
